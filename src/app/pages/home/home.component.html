Angular Handbook

Your personal guide to components, services, architecture, and more.

🚀 Quick Start

  • ng new project-name – Create a new Angular project
  • ng generate component name – Generate a component
  • ng serve – Start dev server

📦 Components

Use components to encapsulate UI logic. Remember:

  • Inputs/Outputs for communication
  • Use `OnPush` when possible
  • Split presentational and container components
